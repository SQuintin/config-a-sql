Datestamp formatting help

Datestamp uses Python strftime format codes. Common examples:

  • %Y%m%d → 20260622
  • %Y-%m-%d → 2026-06-22
  • %m%d%Y → 06222026
  • %Y%m → 202606
  • %Y%m%d_%H%M%S → 20260622_105301

Full Python docs: strftime and strptime format codes .
